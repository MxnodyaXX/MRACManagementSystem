# 🚗 MRAC Management System

A modern and responsive **Vehicle Rental Management System** developed for **Mano Rent a Car Service**.  
This system helps manage vehicles, rentals, owners, and revenue analytics through a professional dashboard interface.

---

# ✨ Features

✅ Vehicle Dashboard  
✅ Revenue Leaderboard  
✅ Available Vehicles Tracking  
✅ Rented Vehicles Monitoring  
✅ Repair & Damaged Vehicle Tracking  
✅ Vehicle Owner Management  
✅ Date Filter Analytics  
✅ Responsive Modern UI  
✅ Fast Performance with Vite  

---

# 🖥️ Dashboard Modules

## 🚘 Vehicle Management
- View all vehicles
- Track vehicle status
- Manage availability
- Vehicle categorization

## 📊 Revenue Analytics
- Top revenue generating vehicles
- Date range filtering
- Performance overview
- Business insights

## 🔧 Repair Management
- Track damaged vehicles
- Maintenance records
- Repair status monitoring

## 👤 Owner Management
- Vehicle owner records
- Owner-vehicle relationships
- Contact management

# 🚀 Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/MxnodyaXX/MRACManagementSystem.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd MRACManagementSystem
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run Development Server

```bash
npm run dev
```
